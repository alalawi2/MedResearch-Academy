## Regenerating Materials

If you need to update any promotional materials (e.g., change date, speaker, or content), use the provided Python scripts:

```bash
cd /home/ubuntu/dr_alawi_website

# Regenerate main flyer
python3 create_flyer.py

# Regenerate Instagram Story
python3 create_instagram_story.py

# Regenerate email signature and social media square
python3 create_email_signature.py
```

Before regenerating, update the text content in the respective Python scripts.
